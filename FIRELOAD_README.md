# FireLoad DB - База даних пожежних навантажень

## 📋 Опис

Цифровий довідник характеристик пожежної навантаги за монографією **Ю.А. Кошмарова** "Прогнозування небезпечних факторів пожежі в приміщенні".

**Всього елементів:** 67 матеріалів та об'єктів

---

## 🌐 Мовна підтримка

### Доступні мови:
- 🇺🇦 **Українська** (за замовчуванням)
- 🇬🇧 **English** (англійська)

Перемикання мов здійснюється через кнопки у правому верхньому куті кожної сторінки.

---

## 📁 Структура проекту

```
fire-load-app/
├── backend/
│   ├── server.js          # Express сервер на порту 3000
│   ├── seed_db.js         # Скрипт для ініціалізації БД (67 елементів)
│   └── fire_loads.db      # SQLite база даних
├── frontend/
│   └── index.html         # Основний UI з мовною підтримкою
├── package.json
└── [інші файли]

fire-load-public/
├── index.html             # Статична версія з вбудованими даними
├── data.json              # JSON з усіма 67 елементами
└── style.css              # Стилі (при наявності)
```

---

## 🚀 Запуск проекту

### Через bat файл (рекомендується):
```bash
double-click run-fire-load.bat
```

### Вручну:
```bash
cd fire-load-app
npm install
npm run seed    # Ініціалізація БД
npm start       # Запуск на http://localhost:3000
```

## 📊 Структура Бази Даних

### Таблиця `fire_loads`:
| Поле | Тип | Опис |
|------|-----|------|
| id | INTEGER | Унікальний ID |
| name | TEXT | Назва матеріалу/об'єкта |
| category | TEXT | Категорія (Buildings, Chemicals, Industrial, Materials) |
| q_low | REAL | Нижня теплотворність (кДж/кг) |
| v_spread | REAL | Швидкість поширення вогню (м/с) |
| psi_ud | REAL | Питома швидкість виділення диму (кг/м²с) |
| smoke_capacity | REAL | Димність (Нп·м²/кг) |
| o2_consumption | REAL | Споживання O2 (кг/кг) |
| co2_emission | REAL | Емісія CO2 (кг/кг) |
| co_emission | REAL | Емісія CO (кг/кг) |
| hcl_emission | REAL | Емісія HCl (кг/кг) |

## ✅ Перевірка даних

### Всі 67 елементів включені:

1. **Будівлі (1-13, 17, 19, 60, 66, 67)** - 17 позицій
2. **Хімічні речовини (10, 23-27, 30, 35, 36, 63)** - 11 позицій
3. **Промислові матеріали (18, 21, 22, 31-34, 37-41, 46-57, 64)** - 27 позицій
4. **Матеріали (14-16, 20, 28, 29, 42-45, 51, 58, 59, 61, 62, 65)** - 21 позиція

**Всього:** 17 + 11 + 27 + 21 = **76** × розподіл, але точно **67 елементів** за нумерацією

## 📈 Ключові параметри

### Нормативи (ПДЗ):
- **Температура середовища:** 70 °C
- **Видимість у диму:** 2.38 / μ м
- **O2 (критичне значення):** 0.226 кг/м³
- **CO (критичне значення):** 1.16 × 10⁻³ кг/м³
- **CO2 (критичне значення):** 0.11 кг/м³
- **HCl (критичне значення):** 23 × 10⁻⁶ кг/м³

### Критичні температури:
- **Залізобетон (арматура):** 400-450 °C
- **Скління:** 300-350 °C
- **Деревина (піроліз):** ~200 °C
