## 📊 Про базу даних

Повна база даних усіх 67 матеріалів та речовин з класичного посібника Ю.А. Кошмарова **"Прогнозування небезпечних факторів пожежі в приміщенні"** з усіма необхідними характеристиками для розрахунків пожежної небезпеки.

### 📈 Параметри

- **Q_н** - Теплота згоряння (кДж/кг)
- **v_п** - Швидкість поширення полум'я (м/(м²·с))
- **ψ_уд** - Димоємність (м²/кг)
- **D_m** - Щільність диму (м²/с)
- **L_O₂** - Споживання кисню (кг/кг)
- **L_CO₂** - Викид CO₂ (кг/кг)
- **L_CO** - Викид CO (кг/кг)
- **L_HCl** - Викид HCl (кг/кг)

## 📦 Версії

### 1️⃣ Static (GitHub Pages) - Рекомендується
- **Файл:** `index.html` 
- **Переваги:** Швидка, без серверу, вбудовані дані
- **Розташування:** Коренева папка

### 2️⃣ Dynamic (Optional Backend)
- **Папка:** `fire-load-app/`
- **Backend:** `backend/server.js` (Node.js)
- **Frontend:** `frontend/index.html`
- **API:** REST на порту 3000

## 🚀 Встановлення (для локального розробки)

### Static версія (GitHub Pages)
```bash
# Просто відкрийте index.html у браузері
start index.html
```

### Dynamic версія (з backend)
```bash
# 1. Встановіть залежності
cd fire-load-app
npm install

# 2. Ініціалізуйте БД
node backend/seed_db.js

# 3. Запустіть сервер
node backend/server.js

# 4. Відкрийте в браузері
start http://localhost:3000/frontend/index.html
```
