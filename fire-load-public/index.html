[ENGLISH] How to Use

1. Select Language: Use UK/EN buttons (top-right)

2. Search: Type substance name in search box to filter results

3. Browse Categories: Materials are grouped into:
• Buildings • Chemicals • Industrial • Materials

4. View Tabs:
• Base - All 67 substances (Koshmakov reference)
• Thresholds - PLE values
• Coefficients - Bashkirtsev constants
• Manual - This guide

[УКРАЇНСЬКА] Як користуватися

1. Оберіть мову: Кнопки UK/EN (верхній правий кут)

2. Пошук: Введіть назву речовини в пошуковому полі

3. Категорії: Матеріали розділені на:
• Будівлі • Хімікати • Промислові • Матеріали

4. Вкладки:
• База - Усі 67 речовин (довідник Кошмарова)
• Нормативи - Значення ПДЗ
• Коефіцієнти - Константи Башкирцева
• Manual - Це керівництво
База даних сформована на основі посібника Ю.А. Кошмарова "Прогнозування небезпечних факторів пожежі в приміщенні".
