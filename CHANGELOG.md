## 🚀 Як запустити

```bash
# Варіант 1: Через bat файл
double-click run-fire-load.bat

# Варіант 2: Вручну
cd fire-load-app
npm install
npm run seed
npm start
# → http://localhost:3000
```

---

## 📌 Примітки

1. База даних очищується та переінціалізується кожен раз при `npm run seed`
2. Мова зберігається в сесії браузера (не персистентна)
3. Пошук чутливий до регістру (lowercase порівняння)
4. Всі параметри мають описи у вкладці "Нормативи"
